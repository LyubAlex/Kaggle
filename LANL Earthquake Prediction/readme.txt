LANL_xgb.ipynb - pipeline for earthquake time prediction using XGBoost
LANL_train.csv - train dataset of 138 features generated using the kernel https://www.kaggle.com/artgor/earthquakes-fe-more-features-and-samples
LANL_test.csv - dataset to make a prediction for submission
features.csv - set of 79 serial numbers of the features selected from the original set of 138 features 
phase1.png - results of the first phase of the optimization procedure for selecting features
